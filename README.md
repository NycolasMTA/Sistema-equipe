## Reflexão
O que é um repositório?
R: É uma pasta oline onde os projetos ficam armazenados.

O que é um commit?
R: Registro de alteração do projeto

O que é uma branch?
R: Uma copia do projeto, para testes.

O que é um Pull Request?
R: Um pedido de mudanças no projeto principal.

Por que usar controle de usuários?
R: Para previnir erros ao sistema principal e causar danos críticos ao projeto.
